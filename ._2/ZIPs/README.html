Here is the formR Repos starter page

From your new Repos folder, you can either

  • Option 1 in a browser:
    1. Download this file, set-repos (use: Save link as...)
    2. Save it as set-repos in a Repos folder (with no extension)
    3. Run it with: bash set-repos
  • Option 2 in a bash terminal:
    • Copy this line of code: curl -s http://aidocs4u.com/set-repos | bash
    • Paste and run it in a terminal.
  • Option 3 in a bash terminal:
    • Copy this line of code: curl -s https://raw.githubusercontent.com/robinmattern/FRTools_prod2-master/master/._2/ZIPs/set-repos | bash
    • Paste and run it in a terminal.

    After that you can run any of these commands:

    • bash install frtools # first
    • bash install anyllm
    • bash install aidocs

    To work on these projects run any of these commands:

    For your own project (use -d to doit)
    1. frt new repo MyProject_dev01
    2. cd MyProject_dev01; code MyProject_dev01.*
    3. gitr add remote MyProject_dev01-bruce {GitHub_Account}
    4. gitr mak remote MyProject_dev01-bruce
    5. Make changes to your code
    6. Use VSCode to commit and sync you changes
    For your AI Model Test project (use -d to doit)
    1. bash install aicoder
    2. gitr clone AI-Tests_dev01-robin
    3. gitr clone AI-Tests_dev01-robin '' final-bruce
    4. gitr clone AI-Tests_dev01-robin '' dev01-bruce
    5. cd AI-Tests_dev01-bruce; code AI-Tests_dev01-bruce.*
    6. gitr add remote AI-Tests_dev01-bruce {GitHub_Account}
    7. gitr mak remote AI-Tests_dev01-bruce
    8. aic list models
    9. ai2code save markdown {ANo} claude.md [{model}]
    10. aic list scripts {ANo} [{model}]
    11. ai2 save scripts {ANo} [{model}]
    12. Use VSCode to test, commit and sync your test model results
    For your AI Model Final project (use -d to doit)
    1. gitr clone AI-Tests_dev01-robin '' final-bruce
    2. cd AI-Tests_dev01-bruce; code AI-Tests_dev01-bruce.*
    3. gitr add remote AI-Tests_final-bruce {GitHub_Account}
    4. gitr mak remote AI-Tests_final-bruce
    5. Use BeyondCompare or WinMerge to copy apps to AI-Tests_final-bruce
    6. Use VSCode to test, commit and sync your final model results
